Easy Voting for 2 College Brothers!

Help support 2 College Brothers Moving & Storage by easily adding a voting tool to your browser. This will help select their three entries on the ballot.

Drag the link below to your Bookmarks Bar (on a desktop browser):

Vote for 2CB (Drag Me!)

Or see instructions below for manual bookmark creation on mobile.

How to Use This Voting Bookmarklet:

  1. On Desktop: Drag the "Vote for 2CB (Drag Me!)" link above directly onto your browser's Bookmarks Bar (also known as the Favorites Bar).
    (If you don't see the Bookmarks Bar, try pressing Ctrl+Shift+B on Windows/ChromeOS or Cmd+Shift+B on Mac).
  2. On Mobile (Manual Creation - more technical):
    • Tap and hold the "Vote for 2CB (Drag Me!)" link above.
    • Choose an option like "Copy Link Address," "Copy URL," or "Copy Link."
    • Open your mobile browser's bookmarks manager (the steps vary by browser).
    • Create a new bookmark. You might need to bookmark the current page first, then edit that new bookmark.
    • For the bookmark's name, enter something like "Vote for 2CB!".
    • Important: For the bookmark's URL/Address field, delete any existing web address and then paste the copied code (it will start with javascript:).
    • Save the bookmark.
  3. Go to the main voting website page: https://tampabaybestofthebest.com/vote/#
  4. Click the "Vote for 2CB (Drag Me!)" bookmarklet that you saved.
  5. The script will attempt to navigate and make selections.
    • If you weren't already on the voting site, the first click will take you there. You'll then need to **click the bookmarklet again** once the voting page has loaded.
  6. After the script runs (it will show an alert: "All actions attempted..."), please review your ballot on the voting site to ensure the correct items are selected, and then click the final "Submit Voting Ballot" button yourself.

This tool automates the selection of three specific entries for 2 College Brothers Moving & Storage to simplify the voting process. You are responsible for reviewing and submitting the final ballot. Make sure all selectors within the bookmarklet code are up-to-date if the website changes.
